## 📋 文档规范

### 文档格式

- 所有文档采用 Markdown 格式编写
- 使用 UTF-8 编码
- 遵循统一的文档模板

### 版本管理

- 文档版本号格式: v{major}.{minor}
- 重大变更增加 major 版本号
- 小幅修改增加 minor 版本号

### 更新流程

1. 创建文档变更请求
2. 团队评审
3. 更新文档内容
4. 更新版本号和变更记录
5. 发布新版本
